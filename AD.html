Formation Active Directory

L’active directory est un système d’annuaire proposé par Microsoft pour les systèmes d’exploitation Windows.

Grâce à cet annuaire, je peux connaitre les unités organisationnelles, les groupes et les ressources. Les UO permettent de structurer notre annuaire, les groupes simplifient la gestion de toutes les ressources qui sont constituées des utilisateurs, imprimantes ou ordinateurs.

Le point fort de l’AD est les stratégies de groupes. Avec ces stratégies, je vais pouvoir mettre de la sécurité au sein de mon entreprise. Grâce aux différents groupes définis, je vais pouvoir appliquer différentes règles avec une portée différente.

L’active directory est la base des serveurs d’une entreprise avec quelques employés. C’est le serveur le mieux protégé car il contient toutes les informations concernant les employés de la boite avec leur mot de passe. Une personne mal intentionnée ayant accès à ce serveur pourra attaquer la société beaucoup plus facilement.

J’ai commencé cette formation en vue du cours de M. Vanlerberghe. Comme il nous avait signalé que ce chapitre était plus complexe, j’ai pris les devants pour être prêt. De plus cette formation m’a aidé pour la semaine d’intégration proposée par l’EPHEC. Je voulais être prêt à mettre en place en active directory.

Je n’ai pas été en mesure de faire l’activité à cause de mon ordinateur qui est limitée pour la RAM. Je devais lancer trois machines en même temps, deux actives directory et une machine Windows 10. Ma machine ramait clairement juste avec deux machines allumées.
